# Instagram Clone by NeoTRAN

## Stack MARN

1- Mongo

2- Apollo

3- React

4- Nodegit

## Download

### Client

git clone https://github.com/NeoTRAN001/Instagram.git

### Run client

cd instagram/client

yarn start

### Run Server

cd instagram/server

yarn dev
